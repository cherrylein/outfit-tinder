Outfit Auswahl App

Oeffnen:
- index.html im Browser oeffnen.

Funktionen:
- Designs nach Kategorie filtern
- Design anklicken und gross ansehen
- Bewertung von 1 bis 5 vergeben
- Favoriten markieren
- Kommentar pro Design notieren
- Bewertungen als JSON oder CSV exportieren

Hinweis:
- Bewertungen werden lokal im Browser gespeichert.
- Wenn du die ZIP an ein Model weitergibst, hat sie eine eigene lokale Bewertung.
- Fuer einen gemeinsamen Stand exportiert ihr am besten JSON oder CSV und schickt die Datei zurueck.
